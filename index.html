HTX support for Sublime Text 2

You'll find Sublime Text 2 over at http://www.sublimetext.com/2.

This package adds support for the HTX syntax found in Visma Zpider. It'll give you autocomplete and syntax highlighting in Sublime Text 2 for HTX files.

Installation

With Package Control (recommended)

First enable Package Control as described at Sublime Package Control

Then add the following repository (Package Manager/Add Repository):

https://raw.github.com/omelhus/HTX/master/packages.json

You'll then be able to install HTX (Visma Zpider) through the Package Control, and it'll autoupdate whenever a new version is pushed.

Manually

Either download with git or download the zip and place it in the Data\Packages folder

git clone https://github.com/omelhus/HTX.git HTX

When you've installed the package, you'll have to make some changes to your settings. Go to Preferences/Settings - User and add these: 

{
	"auto_complete_triggers":
	[
		{
			"characters": "$<",
			"selector": "text.html"
		}
	],
	"word_separators": "./\\()\"'-,.;<>~!@#%^&*|+=[]{}`~?"
}
